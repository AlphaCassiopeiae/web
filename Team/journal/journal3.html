Spent this time researching the S100 bus a bit, and also looked into how programming on the original Altair 8800 works. I did this by reading some of the IEEE standard for the S100 bus and watching some YouTube videos in a playlist called “Altair 8800 Instructional Videos” about how to program on the Altair 8800, which our project is trying to mimic. I learned how the various switches on the front panel operate, including the examine switch for examining memory locations and the deposit switch for putting values in memory. I also read about the voltage levels on the original S100 bus, which were some pretty exotic levels including +/-16V, 8V, and +/-5V. The modifications we will make for our project include running everything on 3.3V and using DDR4 DIMM connectors in place of the original 100-pin connectors, although we will be not be using all 288 pins that the connectors provide (less than 100 probably for our desired functionality). I'll draw a rough sketch of the backplane PCB so it is clearer in my mind, seen below. Since we now have our RP2350 stamp boards, the next step is getting some Rust running on them so we can start emulating CPU/memory/UART for our functionality cards.
