Project Functional Description:

We plan to create a miniaturized version of the Altair 8800, one of the first microcomputers. This device in its minimally complete form will allow the user to:

  1. Utilize the front panel to write, read, and execute Intel 8080 instructions in memory.
  2. Add expansion cards to allow the processor to read and write data over UART, to a disk, etc.

Components

Front Panel

The front panel contains 8 status lights, 16 address bit lights, and 8 data bit lights. It includes switches for CPU control, memory examination, and reset functions.

Internal Cards

The enclosure houses a base PCB with 5 card slots using DDR4 DIMM connectors. It includes CPU, memory, and UART cards, all based on the RP2350 microcontroller.

Operation

Users will use the front panel to write, run, and debug i8080 programs. Programs can be entered manually using switches, stored in memory, and executed via the CPU.

Project Specific Design Requirements (PSDRs):

  1. (Software) Rust-based Intel 8080 processor emulation tied to an external clock.
  2. (Hardware) S-100-inspired 8080 bus protocol via RP2350 PIO with DDR4 DIMM connector.
  3. (Hardware) Switch and LED matrix for the front panel interfacing with the CPU.
  4. (Hardware) 5V to 3.3V step-down power supply using a reference design.
  5. (Hardware) Serial communication via a UART card with custom firmware.
